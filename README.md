# Node.js でつくる WASM トランススレーター

## はじめに

ミニNode.js を、WebAssembly(WASM) に変換するトランスレーターです。
ミニマムなNode.js インタープリターで実行できるように書かれています。

## できること（予定）

- 扱えるのは32ビット符号あり整数のみ
- 四則演算、余りの計算
- ローカル変数
  -  配列やハッシュ（連想配列）は使えない
- if - else
- ループは while のみ
- 組込関数として、整数の出力と、固定文字列の出力ができる
- ユーザ定義関数が使える
- FizzBuzzと、再帰によるフィボナッチ数列の計算がゴール
